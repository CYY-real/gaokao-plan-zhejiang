# 高考志愿填报方案生成 Skill（浙江版）

## 适用场景

用户给出 **浙江高考位次 + 选科组合**，需要生成一份完整的志愿填报方案，包含：
- 专业方向分类推荐（含行业前景、大坑提示、叠加方向建议）
- 省内外院校分层（公办本科/民办本科/专科）
- 冲稳保判定与80个志愿排布
- 优缺点分析与决策建议

## 工作流程

### Phase 1：信息采集

#### 1.1 获取用户信息
```
必须确认：
- 位次（如 120404）
- 总分（约536分）
- 选科组合（如 政治+地理）
- 选科匹配规则（不限 / 思想政治(1门必选) / 地理(1门必选) → 可报；物化 → 排除）
- 一段线（如490分）
- 偏好方向（如 会计/师范/医护/管理）
- 地域偏好（省内/省外/无所谓）
```

#### 1.2 采集核心数据
按以下优先级采集，每项标注来源编号：

| 优先级 | 数据 | 来源 |
|:----:|------|------|
| P0 | 上年（如2025）一段投档线，120404 ± 20000 范围内 | 浙江省教育考试院官网 / 教育在线 / 阳光高考 |
| P0 | 本年（如2026）高职院校招生计划（专科） | KBZY 系统导出的 Excel |
| P0 | 本年省外公办本科在浙招生计划 | KBZY 系统逐校查询 / 各校官网 |
| P1 | 职业技术大学本科招生计划 | KBZY 系统 / 各校官网 |
| P1 | 选科要求确认（不限/政/地） | 各校招生章程 / KBZY 筛选 |
| P2 | 不可选学校清单（位次不够/选科不符） | 从投档线数据中排除 |

#### 1.3 数据整理规则

**位次判定标准（以用户位次为基准）：**
- **冲（🔶）**：位次更高（数值更小），差距在+10000以内
- **稳（🟢）**：位次接近 ±5000
- **保（🟢）**：位次更低（数值更大），差距在-10000以外

**选科排除规则：**
- 物理+化学（物化）捆绑 → ❌ 全部排除
- 物理/化学/生物任选 → ❌ 排除
- 物化生任一 → 按组合判断
- 不限 / 政治 / 地理 → ✅ 可报

### Phase 2：专业方向分析

#### 2.1 按方向分类
对可报专业按以下方向归类：
| 方向 | 典型专业 | 省内代表 | 省外代表 |
|------|---------|---------|---------|
| 💰 会计/财管 | 会计学、财务管理、审计学 | 浙江金融职院、浙经济职院 | 哈商大、山财大 |
| 📚 师范 | 小学教育、学前教育、早期教育 | 金华职业技术大学 | ❌ 位次通常不够 |
| 🏥 医护 | 护理、中医康复、康复治疗 | 浙药科职大、金华职大 | ❌ 通常需物化/生物 |
| 💻 计算机/数字媒体 | 数字媒体技术、大数据技术 | 杭职大、浙金融职院 | ❌ 通常需物化 |
| 📊 管理/电商/外语 | 商务英语、电子商务、物流管理 | 浙金融职院、浙经济职院 | 山财大、哈商大 |

#### 2.2 每个方向输出内容模板
每个方向必须包含3层内容：

**第一层：数据表** — 项目/内容表格
| 项目 | 内容 |
|------|------|
| 推荐行业 | 行业1 + 行业2 + … |
| 省内选择 | 学校·专业（位次，判定） |
| 省外选择 | 学校·专业（位次，判定） |
| 考公考编 | ⭐ 评分 + 说明 |
| 升本路径 | 专科→专升本学校 / 本科一步到位 |
| 核心优势 | 2-3条 |
| 主要风险 | 2-3条 |
| 总体评价 | ⭐⭐⭐⭐⭐ 综合评分 + 一句话总结 |

**第二层：大坑与建议** — 引用块（`> 💡`）
```
> 💡 **[方向]最大的坑是什么？就是[具体痛点——真实就业案例/行业现状]。**
> [2-3句展开，说清楚为什么这是个坑]
>
> **所以你要是选[方向]，我的建议是：[具体行动建议——选什么子方向、大学怎么规划]。**
```

大坑写作要点：
- 用"你"直接对话，口语化
- 必须有真实感（具体数字、真实案例）
- 建议要可执行（具体子方向、具体技能、具体证书）
- 语气像学长/姐姐在说实话，不要官方套话

**第三层：叠加方向推荐** — 表格
| 叠加方向 | 就业去向 | 前景评价 |
|---------|---------|:--------:|
| 主方向 + 热门叠加 | 具体岗位 | ⭐⭐⭐⭐⭐ 评价 + 理由 |
| 主方向 + 新兴叠加 | 具体岗位 | ⭐⭐⭐⭐ 评价 + 理由 |

叠加方向选择原则：
- 优先选"抗AI替代"方向（管理会计 > 基础记账，运动康复 > 普通护理）
- 优先选"杭州/浙江产业优势"方向（跨境电商、直播电商、数字媒体）
- 优先选"低学历门槛"方向（看作品/能力 > 看文凭）
- 新兴方向（ESG、AIGC、银发经济）适当推荐，标注"新兴"

#### 3.2 页面格式约定
- 表格：斑马线（`tr:nth-child(even)`），蓝色表头
- 冲稳保用 🔶🟢 图标标记
- 重点推荐行加粗 `**` （如 **哈尔滨商业大学·会计学**）
- 大坑提示用黄色引用块
- 位次用 `**加粗**` 突出
- 判定列统一：冲/稳/保 + 图标

#### 3.3 参考文献标注规则
- 每个数据章节标题加标注：`## 七、省外公办本科 [1][4][5]`
- 文末附参考资料表格：

## 注意事项

1. **位次是核心**：所有分析以位次为准，分数仅作参考。
2. **选科是硬门槛**：物化绑定专业一律排除，不能有任何侥幸。
3. **平行志愿机制**：浙江省专业+学校平行志愿，按位次投档。本科填在前面不影响专科录取——先录本科，录不上自动落到专科。所以前30冲稳本科+后50保专科是合理策略。
4. **定性结论需谨慎**：行业前景判断、薪资数据、AI替代风险等属于趋势分析，仅供参考。输出时标注"个人建议"。
5. **"大坑"风格**：不要写成官方教条，要有"过来人说实话"的实感。用具体场景、数字、案例说话。
6. **叠加方向**：每个方向必须给出2-5个叠加方向，这是核心增值内容。优先推荐"抗AI替代+低学历门槛+本地产业优势"的组合。
7. **数据过期提醒**：所有位次数据仅代表上年，2026年填报时需以当年数据为准。
