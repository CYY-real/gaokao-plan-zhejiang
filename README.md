# 浙江高考志愿填报方案生成器

> 位次120404 · 选科政治+地理 · 完整方案实战输出

一个面向 **浙江高考生** 的志愿填报方案生成方法论，已沉淀为 opencode/Claude Code 可调用的 Skill。

## 这是什么

本仓库包含一套完整的 **高考志愿填报分析框架**，基于真实位次（120404）和选科组合（政治+地理）实战输出，涵盖：

- **专业方向分类** — 会计/师范/医护/计算机/管理 五大方向，每个含行业前景、大坑提示、叠加方向建议
- **省内外院校分层** — 公办本科/职业技术大学/公办专科 三级筛选
- **冲稳保80志愿排布** — 按位次精准判定
- **优缺点分析** — 省内vs省外、本科vs专科、路径对比

## 使用方法

### 方式一：作为 opencode Skill 使用

1. 将本仓库的 `SKILL.md` 放入 opencode 扫描路径（如 `~/.claude/skills/gaokao-plan/`）
2. 或 在 `opencode.json` 中配置 `"skills": { "paths": ["路径/to/本仓库"] }`
3. 重启 opencode，说："帮我按位次XXX选科YYY生成志愿方案"

Skill 会自动加载方法论框架，按 Phase 1→4 执行。

### 方式二：直接阅读方案

打开 `志愿方案_位次120404_政地.md` 查看完整输出示例，包含数据表、大坑分析、叠加方向推荐、冲稳保排布。

### 方式三：参考模板生成你自己的方案

1. 修改位次和选科
2. 按 SKILL.md 中的 Phase 1 采集最新投档线数据
3. 按 Phase 2-3 组织文档结构
